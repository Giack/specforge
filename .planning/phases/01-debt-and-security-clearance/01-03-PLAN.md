<objective>
Replace the bash heredoc shell injection vector in wave execution with direct `exec.Command` + stdin pipe.

Purpose: The current `executeWithClaude` and `executeWithOpenCode` functions interpolate the AI-generated prompt into a bash script via `fmt.Sprintf`. Any backtick, `$()`, or double-quote in the prompt executes as shell code. With MCP integration coming in Phase 4, this becomes a machine-compromise vector. The fix eliminates the bash wrapper entirely.
Output: `cmd/dev/wave.go` with safe subprocess execution. Shell injection surface is gone.
</objective>
